# simulador-sistema-de-arquivos
Projeto da disciplina de Sistemas Operacionais para a simulação de comandos linux em um sistema de arquivos

# Instruções
Para instalar as dependências necessárias no Linux, caso não tenha
sudo apt update && sudo apt install php php-cli php-pdo php-sqlite3 --nogpgcheck

Para rodar o projeto a partir do diretório /php
php -S localhost:8000

Abrir a página "http://localhost:8000/setup.php" no navegador para criar o banco (se ainda não tiver)
Ir para a página principal fazer cadastro ou login e prosseguir dentro da aplicação
